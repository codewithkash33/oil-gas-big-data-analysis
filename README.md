# oil-gas-big-data-analysis
Big Data analysis of global oil &amp; gas reservoir distribution using Tableau

# Big Data in Oil & Gas Industry
**Author:** Aakash Goyal

## Overview
Analysis of 17,000+ global oil & gas reservoir records using Tableau 
to identify geographic distribution, size patterns, and regional composition.

## Key Findings
- Reservoir distribution is heavily concentrated in specific countries (US, UK, Russia)
- Onshore fields significantly outnumber offshore globally
- Reservoir sizes follow a highly skewed distribution
- North America dominates in total reservoir count

## Tools Used
- Tableau Public (visualization)
- QGIS (geospatial data processing)
- Dataset: OGIM v2.7 via Zenodo
